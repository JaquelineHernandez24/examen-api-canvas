## 📌 Descripción del proyecto
Esta aplicación dibuja, mediante la **API Canvas 2D**, una escena completa compuesta por:

- Fondo con cielo degradado  
- Suelo texturizado  
- Casa con techo y ventanas  
- Árbol volumétrico  
- Sol con halo y rayos  
- Nubes redondeadas  
- Un gato geométrico con más de 30 figuras  
- Más de **60 figuras básicas** entre círculos, rectángulos y triángulos  

El objetivo es recrear una imagen original con estilo artístico utilizando programación gráfica.
